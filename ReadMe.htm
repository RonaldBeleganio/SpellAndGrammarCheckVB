Title: Spelling and Grammar Checker.

This sample demonstrates how to add spelling and grammar checking functions into VB.NET applications. It employs a COM wrapper to import spelling and grammar components into VB.NET application. 

Requirements

  • Microsoft Word 97, 2000, XP (or later) 
  • Trial or Release version of Visual Studio .NET Professional (or later)

Please note: This sample is based on Microsoft Word 2002 and references Microsoft Word 10.0 Object Library (version 8.2). If you do not have the same version, please remove Microsoft Word 10.0 Object Library reference from the project and include the appropriate one. (Read section "Adding reference to Microsoft Word Object Library").

Running the Sample

Type some text in the textbox and press "Spelling Check" or "Grammar Check" as required.

"Spelling Check" only checks spelling, whereas "Grammar Check" checks the grammar and spellings.

Adding reference to Microsoft Word Object Library

  • Right click on References under the Windows Application in the Solution Explorer.  
  • Click on Add Reference...
  • Click the COM tab in the Add Reference dialog box.
  • Scroll down the list until you find the appropriate Microsoft Word Object Library.
  • Click the Select button to select this object.
  • Press OK button.
